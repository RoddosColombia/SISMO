**async def emit(self, event: RoddosEvent) -> None:**
Per D-01: publish-only. Per D-04: enforce agent permissions. Per D-05: idempotent.

Steps:
1. Call `validate_write_permission(event.source_agent, "roddos_events")` — raises PermissionError immediately if denied (NOT sent to DLQ per D-06).
2. Auto-populate `event.label` from `EVENT_LABELS.get(event.event_type, event.event_type)` if label is empty.
3. Try `await self.db.roddos_events.insert_one(event.to_mongo())`.
4. On `pymongo.errors.DuplicateKeyError` — silently return (idempotent, per D-05/BUS-02).
5. On any other Exception (network, timeout) — call `await self._send_to_dlq(event, error)` then return without raising (per D-06/BUS-03).

**async def _send_to_dlq(self, event: RoddosEvent, error: Exception) -> None:**
Per D-08: persist to roddos_events_dlq collection.

1. Compute `next_retry` as `datetime.now(timezone.utc) + timedelta(minutes=5)` (first retry in 5 min).
2. Create `DLQEvent(event_id=event.event_id, event_type=event.event_type, timestamp_utc=event.timestamp_utc, source_agent=event.source_agent, payload=event.payload, correlation_id=event.correlation_id, original_actor=event.actor, error_message=str(error), retry_count=0, next_retry=next_retry.isoformat())`.
3. Try `await self.db.roddos_events_dlq.insert_one(dlq_event.to_mongo())`.
4. On any exception — log error with `logger.error(...)`, do NOT raise (never block caller).

**async def retry_dlq(self) -> int:**
Per D-07: exponential backoff: 5min, 15min, 45min, 2h, 6h. Max 5 retries.

1. Query: `self.db.roddos_events_dlq.find({"retry_count": {"$lt": 5}, "next_retry": {"$lte": now_iso}})`.
2. For each DLQ doc:
   a. Reconstruct a minimal dict from DLQ fields and try `await self.db.roddos_events.insert_one(...)`.
   b. On success: delete the DLQ doc (`delete_one({"event_id": doc["event_id"]})`).
   c. On failure: increment retry_count, compute next_retry using backoff list `[5, 15, 45, 120, 360]` minutes, update DLQ doc. If retry_count >= 5, set `next_retry = None` (permanently failed).
3. Return count of successfully retried events.

**async def get_bus_health(self) -> dict:**
Per D-11/BUS-05: live MongoDB aggregation.

1. `dlq_pending = await self.db.roddos_events_dlq.count_documents({"retry_count": {"$lt": 5}})`.
2. One hour ago: `one_hour_ago = (datetime.now(timezone.utc) - timedelta(hours=1)).isoformat()`.
3. `events_last_hour = await self.db.roddos_events.count_documents({"timestamp_utc": {"$gte": one_hour_ago}})`.
4. Determine status: `"healthy"` if dlq_pending == 0, `"degraded"` if dlq_pending > 0 and dlq_pending < 10, `"down"` if dlq_pending >= 10.
5. Return `{"dlq_pending": dlq_pending, "events_last_hour": events_last_hour, "status": status}`.

```python
async def _retry_dlq_events():
    """Retry failed DLQ events every 5 minutes (per D-07/BUS-04)."""
    from database import db
    from services.event_bus_service import EventBusService
    bus = EventBusService(db)
    try:
        retried = await bus.retry_dlq()
        if retried > 0:
            logger.info("[Scheduler] DLQ retry: %d events re-published", retried)
    except Exception as e:
        logger.error("[Scheduler] DLQ retry failed: %s", e)
```
